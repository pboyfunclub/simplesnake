#SIMPLE SNAKE GAME

This is inspired by Snake game on small Nokia mobile phones.
I built this just for fun and contributions are more than welcome.

##Compatibility

Currently this game is only compatible with Unix and all Unix-like systems

##How to ?

###Compile and Run

make 

./snake
